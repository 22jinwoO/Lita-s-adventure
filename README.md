# 📌 주요 기능
구현기능:

- 캐릭터 컨트롤러를 이용한 플레이어 이동
- 몬스터 유한상태머신,
- Json을 이용한 데이터 저장 및 불러오기, 인벤토리 아이템 추가, 판매
- 씬 이동
- 오브젝트 풀링
- 스크랩터블 오브젝트(아이템 정보, 몬스터 정보)
